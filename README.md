# opengl-sandbox

WIP.

experiment environment for realtime graphics methods in OpenGL.

## Requirements

* C++17
* CMake 3.12 or Higher
* OpenGL 3.3
* [assimp](https://github.com/assimp/assimp)

## Setup

```bash
git submodule update --init
```

## Build

```bash
mkdir build
cd build
cmake ..
make
```
